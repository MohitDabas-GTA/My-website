Complete HTML Tutorial

Master HTML from the basics to advanced concepts with practical examples, exercises and real-world projects.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages. HTML tells the browser how content should be structured. Using HTML you can create:

  • Web pages
  • Blogs
  • Business websites
  • Landing pages
  • Portfolio websites
  • Online stores

Every website on the internet uses HTML as its foundation.

Lesson 1

HTML Document Structure

Every HTML page follows a standard structure. It tells the browser how to display your webpage correctly. 

<!DOCTYPE html>
<html lang="en">

<head>

    <meta charset="UTF-8">
    <title>My First Webpage</title>

</head>

<body>

    <h1>Hello World</h1>

</body>

</html>

Explanation

  • <!DOCTYPE html> tells the browser to use HTML5.
  • <html> is the root element of the webpage.
  • <head> stores metadata like the title and CSS links.
  • <body> contains everything visible on the webpage.
Lesson 2

HTML Headings

HTML provides six heading levels. They help organize content and make pages easier to read. 

<h1>Main Heading</h1>
<h2>Section Heading</h2>
<h3>Sub Heading</h3>
<h4>Smaller Heading</h4>
<h5>Small Heading</h5>
<h6>Smallest Heading</h6>

Use only one <h1> on a page because it represents the main topic. Then use <h2>, <h3>, and so on to organize the remaining content.

Lesson 3

Paragraphs

Paragraphs are created using the <p> element. 

<p>Welcome to DevNova.</p>

<p>This is another paragraph.</p>

Browsers automatically add space before and after each paragraph, making your content easier to read.

Lesson 4

HTML Comments

Comments are ignored by the browser. They help explain your code and make it easier to maintain. 

<!-- This is a comment -->

<h1>Hello World</h1>

Use comments to document sections of your code or temporarily disable HTML during development.

Lesson 5

HTML Links

Links connect one webpage to another. They are created using the <a> (anchor) element. 

<a href="https://example.com">
Visit Example
</a>

The href attribute specifies the destination URL.

Open Link in New Tab

<a href="https://example.com"
target="_blank">

Open Website

</a>

Common Attributes

  • href – Link destination
  • target – Where to open the link
  • title – Tooltip text
  • download – Download a file
Lesson 6

HTML Images

Images are displayed using the <img> element. 

<img src="images/coding.jpg"

alt="Person Coding"

width="500">

Important Attributes

  • src — Image path
  • alt — Alternative text for accessibility
  • width — Image width
  • height — Image height

Always provide meaningful alt text. It improves accessibility and helps search engines understand the image.

Lesson 7

HTML Lists

HTML supports unordered, ordered and description lists.

Unordered List

<ul>

    <li>HTML</li>

    <li>CSS</li>

    <li>JavaScript</li>

</ul>

Ordered List

<ol>

    <li>Install VS Code</li>

    <li>Write HTML</li>

    <li>Open Browser</li>

</ol>
Lesson 8

HTML Tables

Tables organize data into rows and columns. 

<table>

<tr>

    <th>Language</th>

    <th>Difficulty</th>

</tr>

<tr>

    <td>HTML</td>

    <td>Easy</td>

</tr>

<tr>

    <td>JavaScript</td>

    <td>Medium</td>

</tr>

</table>

Table Elements

  • <table> — Creates a table
  • <tr> — Table row
  • <th> — Table header
  • <td> — Table cell
Lesson 9

HTML Text Formatting

HTML provides several elements to format text. 

<strong>Bold Text</strong>

<em>Italic Text</em>

<mark>Highlighted</mark>

<small>Small Text</small>

<del>Deleted Text</del>

<ins>Inserted Text</ins>

<sub>Subscript</sub>

<sup>Superscript</sup>

Prefer using semantic elements like <strong> and <em> because they add meaning as well as styling.

Lesson 10

HTML Forms

HTML forms allow users to send information to a website. Forms are commonly used for login pages, registration, contact forms, surveys, and search boxes. 

<form action="/submit" method="post">

    <label>Name</label>
    <input type="text" name="name">

    <label>Email</label>
    <input type="email" name="email">

    <button type="submit">Submit</button>

</form>

The action attribute specifies where the data is sent, while the method attribute specifies how it is sent.

Lesson 11

HTML supports many different input types for collecting user data.

Input Type Purpose
text Single-line text
password Password field
email Email address
number Numeric input
date Select a date
checkbox Multiple choices
radio Single choice
file Upload files
color Choose a color
Lesson 12

Labels and Textareas

Labels improve accessibility by connecting descriptive text with form controls. 

<label for="message">

Message

</label>

<textarea

id="message"

rows="5"

cols="40">

</textarea>

Using labels makes forms easier to use, especially for screen readers.

Lesson 13

Dropdown Lists

Use the <select> element to create a dropdown menu. 

<label>Choose Language</label>

<select>

    <option>HTML</option>

    <option>CSS</option>

    <option>JavaScript</option>

    <option>Python</option>

</select>
Lesson 14

Semantic HTML

Semantic elements describe the meaning of the content instead of only its appearance. They improve accessibility, readability, and search engine optimization. 

<header></header>

<nav></nav>

<main></main>

<section></section>

<article></article>

<aside></aside>

<footer></footer>

Benefits

  • Better SEO
  • Cleaner code
  • Improved accessibility
  • Easier maintenance
Lesson 15

HTML5 Elements

HTML5 introduced many new semantic elements and multimedia features that make modern web development easier.

  • <header>
  • <footer>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <figure>
  • <figcaption>
  • <audio>
  • <video>
  • <canvas>

Using these elements correctly creates more organized and maintainable web pages.

Lesson 16

HTML Audio

HTML5 allows you to add audio to your webpages without using external plugins. 

<audio controls>

    <source src="music.mp3" type="audio/mpeg">

    Your browser does not support the audio element.

</audio>

Common Attributes

  • controls - Shows play and pause controls.
  • autoplay - Starts playing automatically.
  • loop - Repeats the audio.
  • muted - Starts with the sound muted.
Lesson 17

HTML Video

Videos can be embedded directly into webpages using the <video> element. 

<video width="600" controls>

    <source src="video.mp4" type="video/mp4">

    Your browser does not support video.

</video>

Supported video formats include MP4, WebM, and Ogg.

Lesson 18

HTML Iframes

An iframe displays another webpage inside your current webpage. 

<iframe

src="https://example.com"

width="800"

height="400"

title="Example Website">

</iframe>

Always include a meaningful title attribute to improve accessibility.

Lesson 19

HTML Meta Tags

Meta tags provide information about a webpage to browsers and search engines. 

<meta charset="UTF-8">

<meta name="viewport"

content="width=device-width, initial-scale=1.0">

<meta name="description"

content="Learn HTML from scratch.">

Why Meta Tags Matter

  • Improve SEO
  • Support responsive design
  • Help browsers understand your page
  • Improve search engine snippets
Lesson 20

HTML for SEO

Search Engine Optimization (SEO) helps search engines understand your content and makes it easier for people to find your website.

SEO Best Practices

  • Use one descriptive <h1> per page.
  • Write clear page titles.
  • Add useful meta descriptions.
  • Use semantic HTML elements.
  • Write meaningful alt text for images.
  • Create descriptive link text.
  • Organize headings logically.
  • Keep your HTML clean and valid.

Good SEO improves the user experience as well as search engine visibility.

Lesson 21

Accessibility

Accessibility ensures that websites can be used by as many people as possible, including users with disabilities.

Accessibility Tips

  • Use semantic HTML.
  • Add descriptive alt text to images.
  • Associate labels with form fields.
  • Maintain good color contrast.
  • Make websites keyboard-friendly.
  • Use clear and descriptive link text.

Building accessible websites improves usability for everyone.

Lesson 22

HTML Best Practices

  • Write clean and properly indented code.
  • Use lowercase element and attribute names.
  • Close all required tags correctly.
  • Keep your code organized.
  • Use semantic HTML whenever possible.
  • Avoid unnecessary nested elements.
  • Validate your HTML regularly.
  • Optimize images before uploading.
  • Use external CSS and JavaScript files.
  • Test your pages on multiple devices.

Following these practices will help you build faster, cleaner, and more maintainable websites.

Lesson 23

Mini Project: Build a Simple Portfolio Page

The best way to learn HTML is by creating a real webpage. Here's a simple portfolio layout that combines many of the elements you've learned. 

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>My Portfolio</title>
</head>

<body>

    <header>
        <h1>John Doe</h1>
        <p>Web Developer</p>
    </header>

    <section>
        <h2>About Me</h2>
        <p>I love creating websites.</p>
    </section>

    <section>
        <h2>Skills</h2>
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ul>
    </section>

    <footer>
        © 2026 John Doe
    </footer>

</body>

</html>

FAQ

Frequently Asked Questions

Can I learn HTML without programming experience?

Yes. HTML is one of the easiest technologies to learn and is an excellent starting point for beginners.

How long does it take to learn HTML?

Most beginners can understand the basics within a few days of consistent practice. Becoming comfortable with HTML typically takes several weeks of building projects.

Should I learn CSS after HTML?

Yes. HTML provides the structure of a webpage, while CSS controls its appearance. They are commonly learned together.
