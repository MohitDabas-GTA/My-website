Complete Python Tutorial

Learn Python programming from scratch with beginner-friendly lessons, real-world examples, coding exercises, projects, quizzes, interview questions, and best practices.

What is Python?

Python is a powerful, high-level programming language known for its simple syntax and readability. It is used for web development, automation, data science, artificial intelligence, cybersecurity, software development, and much more.

Python is popular because it is:

  • Easy to learn
  • Readable and beginner-friendly
  • Cross-platform
  • Open source
  • Supported by a large community
  • Used by many companies worldwide
Lesson 1

Installing Python

Download the latest version of Python from the official website and follow the installation instructions for your operating system. 

python --version

# or

python3 --version

After installation, verify that Python is correctly installed by running the version command in your terminal or command prompt.

Your First Python Program

The traditional first program prints a message to the screen. 

print("Hello, DevNova!")

The print() function displays output in the console.

Lesson 3

Variables

Variables store data that can be used throughout your program. 

name = "Alice"

age = 21

is_student = True

print(name)
print(age)

Variable Naming Rules

  • Use descriptive names.
  • Variable names are case-sensitive.
  • Use letters, numbers, and underscores.
  • Do not start a variable name with a number.

Input and Output

Python uses the input() function to receive user input and the print() function to display output.

age = int(input("Enter your age: "))

print(age + 5)

Loops

Loops repeat code automatically until a condition becomes false. 

for i in range(1,6):

Lesson 17

List Comprehensions

List comprehensions provide a concise way to create new lists. 

squares = [

x * x

for x in range(1,6)

]

print(squares)

even = [

x

for x in range(20)

if x % 2 == 0

]

print(even)

List comprehensions are often faster and more readable than traditional loops.

Lesson 18

File Handling

Python can read from and write to files. Always close files after use or use the with statement, which closes the file automatically.

Reading a File

with open("notes.txt", "r") as file:

    content = file.read()

    print(content)

Writing to a File

with open("notes.txt", "w") as file:

    file.write("Welcome to DevNova!")

Exception Handling

Exceptions are runtime errors. Python uses try and except to handle them. 

try:

    number = int(input("Enter a number: "))

    print(number)

except ValueError:

    print("Please enter a valid number.")

Common Exception Blocks

  • try
  • except
  • else
  • finally

Lesson 22

Classes and Objects

Object-Oriented Programming (OOP) organizes code into classes and objects. 

class Student:

    def __init__(self, name):

        self.name = name

student = Student("Alice")

print(student.name)

A class is a blueprint, and an object is an instance of that blueprint.

Lesson 23

Inheritance

Inheritance allows one class to reuse the properties and methods of another. 

class Animal:

    def speak(self):

        print("Animal Sound")

class Dog(Animal):

    def bark(self):

        print("Woof!")

dog = Dog()

dog.speak()

dog.bark()
Lesson 24

Virtual Environments

Virtual environments create isolated Python environments for each project, helping manage dependencies and avoid version conflicts. 

python -m venv myenv

# Windows

myenv\Scripts\activate

# macOS/Linux

source myenv/bin/activate

pip install requests

Using virtual environments is considered a best practice for Python development.

Lesson 25

Lambda Functions

Lambda functions are small anonymous functions that can be written in a single line. 

square = lambda x: x * x

print(square(5))

add = lambda a, b: a + b

print(add(10, 20))

Lambda functions are commonly used with functions such as map(), filter(), and sorted().

Lesson 26

Iterators

An iterator allows you to loop through data one element at a time. 

numbers = [10,20,30]

iterator = iter(numbers)

print(next(iterator))

print(next(iterator))

print(next(iterator))

Many Python objects such as lists, tuples, strings, and dictionaries are iterable.

Lesson 27

Generators

Generators produce values one at a time using the yield keyword. 

def numbers():

    yield 1

    yield 2

    yield 3

for n in numbers():

    print(n)

Generators are memory-efficient because they generate values only when needed.

Lesson 32

Python Best Practices

Following good coding practices makes programs easier to understand and maintain.

  • Use meaningful variable names.
  • Follow PEP 8 style guidelines.
  • Keep functions short and focused.
  • Use virtual environments for projects.
  • Write comments only where needed.
  • Handle exceptions properly.
  • Reuse code with functions and classes.
  • Test your programs regularly.
  • Organize code into modules.
  • Keep learning through real projects.
Project

Mini Project - Number Guessing Game

This simple project generates a random number, and the user has to guess it. 

import random

secret = random.randint(1,10)

guess = int(input("Guess a number (1-10): "))

if guess == secret:

    print("Congratulations! You guessed correctly.")

else:

    print("Wrong guess.")

    print("The number was:", secret)

This project teaches variables, user input, conditions, and the random module.

Common Beginner Mistakes

  • Incorrect indentation.
  • Forgetting the colon (:) after if, for, and while.
  • Using undefined variables.
  • Ignoring exception handling.
  • Not closing files properly.
  • Confusing lists and tuples.
