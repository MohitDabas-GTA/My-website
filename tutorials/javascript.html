Complete JavaScript Tutorial

Learn JavaScript from scratch with beginner-friendly lessons, interactive examples, projects, quizzes, interview questions, and modern ES6 features.

What is JavaScript?

JavaScript is a programming language that makes websites interactive. It allows you to create dynamic web pages, validate forms, update content without reloading the page, create animations, and build complete web applications.

JavaScript is one of the three core technologies of the web:

  • HTML — Structure
  • CSS — Styling
  • JavaScript — Functionality
Lesson 1

Adding JavaScript

JavaScript can be added in three different ways.

Inline JavaScript

<button onclick="alert('Hello!')">

Click Me

</button>

Internal JavaScript

<script>

console.log("Hello World");

</script>

External JavaScript (Recommended)

<script src="script.js"></script>

Using an external JavaScript file keeps your code organized and easier to maintain.

Lesson 2

Variables

Variables are used to store information that can be reused later in your program. 

let name = "DevNova";

const year = 2026;

var message = "Hello";

Variable Keywords

  • let — Block scoped and can be updated.
  • const — Block scoped and cannot be reassigned.
  • var — Function scoped (older syntax, generally avoided in new code).
Lesson 3

JavaScript Data Types

JavaScript supports several built-in data types for storing different kinds of information. 

let name = "John";

let age = 20;

let isStudent = true;

let city = null;

let score;

let person = {

name:"John",

age:20

};

let colors = ["Red","Blue","Green"];

Primitive Data Types

  • String
  • Number
  • Boolean
  • Undefined
  • Null
  • BigInt
  • Symbol
Lesson 4

JavaScript Operators

Operators perform calculations and comparisons. 

let a = 10;

let b = 5;

console.log(a+b);

console.log(a-b);

console.log(a*b);

console.log(a/b);

console.log(a%b);

Operator Categories

  • Arithmetic (+ - * / %)
  • Assignment (= += -=)
  • Comparison (== === != > <)
  • Logical (&& || !)
Lesson 5

Conditional Statements

Conditional statements execute code based on a condition. 

let age = 18;

if(age >= 18){

console.log("Adult");

}else{

console.log("Minor");

}

Conditional Statements

  • if
  • else
  • else if
  • switch
Lesson 6

Loops

Loops repeat a block of code multiple times. 

for(let i=1;i<=5;i++){

console.log(i);

}

let i=1;

while(i<=5){

console.log(i);

i++;

}

Common Loops

  • for
  • while
  • do...while
  • for...of
  • for...in
Lesson 7

Functions

Functions group reusable code into a single block. 

function greet(name){

return "Hello " + name;

}

console.log(greet("DevNova"));

Function Benefits

  • Reusable code
  • Easier maintenance
  • Cleaner programs
  • Improved readability
Lesson 8

Arrow Functions (ES6)

Arrow functions provide a shorter syntax for writing functions. 

const add = (a,b)=>{

return a+b;

};

console.log(add(5,3));

Arrow functions are commonly used in modern JavaScript applications, especially with array methods and frameworks like React.

JavaScript Arrays

An array stores multiple values in a single variable. 

const languages = [

"HTML",

"CSS",

"JavaScript",

"Python"

];

console.log(languages[0]);

console.log(languages.length);

Common Array Methods

  • push()
  • pop()
  • shift()
  • unshift()
  • includes()
  • indexOf()
Lesson 10

JavaScript Objects

Objects store related data using key-value pairs. 

const student = {

name:"John",

age:20,

course:"Web Development"

};

console.log(student.name);

console.log(student.course);

Objects are widely used for storing structured data.

Lesson 11

String Methods

JavaScript provides useful methods for working with text. 

let text = "Hello DevNova";

console.log(text.length);

console.log(text.toUpperCase());

console.log(text.toLowerCase());

console.log(text.includes("Dev"));

console.log(text.replace("Hello","Welcome"));

Useful Methods

  • length
  • toUpperCase()
  • toLowerCase()
  • trim()
  • replace()
  • includes()
  • slice()
Lesson 12

Useful Array Methods

Modern JavaScript includes powerful methods for working with arrays. 

const numbers = [1,2,3,4,5];

numbers.forEach(num => console.log(num));

const doubled = numbers.map(num => num * 2);

const even = numbers.filter(num => num % 2 === 0);

console.log(doubled);

console.log(even);

Popular Methods

  • forEach()
  • map()
  • filter()
  • find()
  • reduce()
  • sort()
Lesson 13

Introduction to the DOM

The DOM (Document Object Model) represents the HTML page as objects that JavaScript can access and modify. 

document.body.style.background = "#f4f4f4";

Using the DOM, JavaScript can update content, styles, and page structure without reloading the page.

Lesson 14

Selecting HTML Elements

Before changing an element, JavaScript must first select it. 

document.getElementById("title");

document.getElementsByClassName("card");

document.querySelector(".btn");

document.querySelectorAll("p");

Selection Methods

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
Lesson 15

Changing HTML Content

JavaScript can update webpage content dynamically. 

const heading = document.querySelector("h1");

heading.textContent = "Welcome to DevNova";

heading.style.color = "#2563eb";

This technique is commonly used for interactive applications and live content updates.

Lesson 16

JavaScript Events

Events allow JavaScript to respond to user actions such as clicking a button, typing in a form, or moving the mouse. 

const button = document.querySelector("#btn");

button.addEventListener("click", function(){

alert("Button Clicked!");

});

Common Events

  • click
  • dblclick
  • mouseover
  • mouseout
  • keydown
  • keyup
  • submit
  • change
  • input
Lesson 17

Form Validation

Form validation ensures that users enter valid data before submitting a form. 

const form = document.querySelector("#signup");

form.addEventListener("submit", function(event){

const name = document.querySelector("#name").value;

if(name === ""){

event.preventDefault();

alert("Name is required.");

}

});

Validation improves user experience and helps prevent incorrect data from being submitted.

Lesson 18

setTimeout()

The setTimeout() function runs code once after a specified delay. 

setTimeout(function(){

console.log("Hello after 3 seconds");

},3000);

This function is useful for notifications, delayed actions, and loading effects.

Lesson 19

setInterval()

The setInterval() function repeatedly runs code at a fixed time interval. 

setInterval(function(){

console.log("Running...");

},1000);

Use clearInterval() to stop a repeating interval when it is no longer needed.

Popular ES6 Features

Lesson 21

Destructuring

Destructuring extracts values from arrays or properties from objects into separate variables. 

const person = {

name:"John",

age:22

};

const {name, age} = person;

console.log(name);

console.log(age);

Destructuring makes code shorter and easier to read.

Lesson 22

JavaScript Modules

Modules allow you to split your JavaScript code into multiple reusable files. 

// math.js

export function add(a,b){

return a+b;

}

// app.js

import { add } from "./math.js";

console.log(add(5,3));

Using modules keeps projects organized and makes code easier to maintain.

Lesson 23

JavaScript Promises

A Promise represents the eventual completion (or failure) of an asynchronous operation. 

const myPromise = new Promise((resolve, reject) => {

let success = true;

if(success){

resolve("Data loaded successfully.");

}else{

reject("Something went wrong.");

}

});

myPromise

.then(result => console.log(result))

.catch(error => console.log(error));

Promises help avoid deeply nested callback functions and make asynchronous code easier to read.

Lesson 24

Async and Await

The async and await keywords simplify working with asynchronous code. 

async function loadData(){

return "Welcome to DevNova";

}

loadData().then(data => console.log(data));

async function getMessage(){

const message = await loadData();

console.log(message);

}

getMessage();
Lesson 25

Fetch API

The Fetch API allows JavaScript to request data from a server without reloading the page. 

fetch("https://jsonplaceholder.typicode.com/posts/1")

.then(response => response.json())

.then(data => console.log(data))

.catch(error => console.log(error));

The Fetch API is commonly used to retrieve data from REST APIs.

Lesson 26

Working with JSON

JSON (JavaScript Object Notation) is a lightweight format used to exchange data between applications. 

const user = {

name:"John",

age:25

};

const json = JSON.stringify(user);

console.log(json);

const obj = JSON.parse(json);

console.log(obj.name);
Lesson 27

Local Storage

Local Storage lets you save data in the browser so it remains available after the page is refreshed. 

localStorage.setItem("username","DevNova");

const user = localStorage.getItem("username");

console.log(user);

localStorage.removeItem("username");

Local Storage is useful for user preferences, themes, and small amounts of persistent data.

Lesson 28

Error Handling

JavaScript errors can be handled using try...catch to prevent your application from crashing. 

try{

let result = 10 / 0;

console.log(result);

}catch(error){

console.log(error);

}

Handling errors gracefully makes applications more reliable and easier to debug.

Lesson 29

Debugging JavaScript

Debugging is the process of finding and fixing errors in your code. 

let total = 50;

console.log(total);

debugger;

Useful Debugging Tools

  • console.log()
  • console.table()
  • console.error()
  • Browser Developer Tools
  • Breakpoints
Lesson 30

JavaScript Best Practices

Following best practices makes your code cleaner, more maintainable, and easier to understand.

  • Use const by default and let when values change.
  • Write meaningful variable and function names.
  • Keep functions small and focused on one task.
  • Comment only when necessary.
  • Avoid global variables.
  • Handle errors using try...catch.
  • Use strict equality (===) instead of ==.
  • Organize code into modules.
  • Test your code thoroughly.
  • Keep learning modern JavaScript features.
Project

Mini Project - Digital Counter

Let's build a simple counter application using JavaScript. 

<button onclick="increase()">+</button>

<h2 id="count">0</h2>

<button onclick="decrease()">-</button>

<script>

let count = 0;

function increase(){

count++;

document.getElementById("count").textContent = count;

}

function decrease(){

count--;

document.getElementById("count").textContent = count;

}

</script>

This project teaches variables, functions, DOM manipulation, and events.

Lesson 31

DOM Project - Change Theme

The following example changes the webpage background color when a button is clicked. 

<button id="themeBtn">

Toggle Theme

</button>

<script>

const btn = document.getElementById("themeBtn");

btn.addEventListener("click",()=>{

document.body.classList.toggle("dark");

});

</script>
