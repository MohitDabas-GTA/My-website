Complete CSS Tutorial

Learn CSS from the ground up and discover how to create beautiful, responsive, and modern websites with real-world examples.

What is CSS?

CSS stands for Cascading Style Sheets. It is the language used to style HTML documents and control how webpages look.

With CSS you can:

  • Change colors
  • Choose fonts
  • Create layouts
  • Add animations
  • Make websites responsive
  • Build professional user interfaces

Without CSS, every webpage would appear as plain text with very limited visual styling.

Lesson 1

Adding CSS to HTML

There are three common ways to apply CSS to a webpage.

1. Inline CSS

<h1 style="color: blue;">

Hello World

</h1>

2. Internal CSS

<style>

h1{

color:blue;

}

</style>

3. External CSS (Recommended)

<link rel="stylesheet"

href="style.css">

External CSS is the preferred approach because it keeps styling separate from HTML, making your code easier to maintain.

Lesson 2

CSS Syntax

A CSS rule consists of a selector and one or more declarations. 

h1{

color:blue;

font-size:40px;

}

Parts of a CSS Rule

  • Selector — Chooses the HTML element.
  • Property — Specifies what to style.
  • Value — Defines how it should appear.

  • Build professional user interfaces

CSS Syntax

h1{

color:blue;

font-size:40px;

Parts of a CSS Rule

  • Value — Defines how it should appear.
Lesson 3

CSS Selectors

Selectors tell CSS which HTML elements should receive a particular style. 

h1{

color:blue;

}

.title{

color:red;

}

#header{

background:black;

}

Types of Selectors

  • Element Selector – Selects HTML elements like h1 or p.
  • Class Selector (.class) – Used on multiple elements.
  • ID Selector (#id) – Used for one unique element.
  • Universal Selector (*) – Selects every element.
  • Group Selector (,) – Styles multiple elements together.
Lesson 4

CSS Colors

CSS supports many ways to specify colors. 

h1{

color:red;

}

body{

background:#f4f4f4;

}

button{

background:rgb(0,120,255);

color:white;

}

Color Formats

  • Named Colors (red, blue, green)
  • HEX (#ffffff)
  • RGB
  • RGBA
  • HSL
  • HSLA
Lesson 5

Backgrounds

Background properties let you customize the appearance of elements. 

body{

background-color:#fafafa;

background-image:url("bg.jpg");

background-repeat:no-repeat;

background-size:cover;

}

You can use colors, images, gradients, or a combination of these to create attractive designs.

Lesson 6

Borders

Borders surround an element and help separate content visually. 

.box{

border:2px solid black;

border-radius:10px;

}

Common Border Styles

  • solid
  • dashed
  • dotted
  • double
  • groove
  • ridge
Lesson 7

Margin and Padding

Margin creates space outside an element, while padding creates space inside the element. 

.card{

margin:20px;

padding:30px;

}
Property Description
margin Outside spacing
padding Inside spacing
Lesson 8

The CSS Box Model

Every HTML element is treated as a rectangular box consisting of:

  • Content
  • Padding
  • Border
  • Margin
.box{

width:300px;

padding:20px;

border:2px solid #333;

margin:30px;

}

Understanding the box model is essential for building consistent and responsive layouts.

Lesson 15

CSS Grid Layout

CSS Grid is a two-dimensional layout system that allows you to arrange elements into rows and columns with ease. 

.container{

display:grid;

grid-template-columns:repeat(3,1fr);

gap:20px;

}

Common Grid Properties

  • display:grid
  • grid-template-columns
  • grid-template-rows
  • gap
  • grid-column
  • grid-row

Grid is ideal for dashboards, galleries, portfolios, and complex page layouts.

Lesson 16

CSS Transitions

Transitions create smooth animations whenever a CSS property changes. 

button{

background:#2563eb;

color:white;

transition:0.3s;

}

button:hover{

background:#1d4ed8;

}

Transitions improve user experience by making interactions feel more natural.

Lesson 17

CSS Animations

Animations allow elements to move, fade, rotate, or scale without using JavaScript. 

@keyframes fadeIn{

from{

opacity:0;

}

to{

opacity:1;

}

}

.box{

animation:fadeIn 1s ease;

}

Use animations sparingly to improve the interface without distracting users.

Lesson 18

CSS Transform

The transform property changes the position, size, or rotation of an element. 

.card:hover{

transform:scale(1.05);

}

.image:hover{

transform:rotate(5deg);

}

Common Transform Functions

  • scale()
  • rotate()
  • translate()
  • skew()
Lesson 19

Responsive Design

Responsive design ensures your website works well on desktops, tablets, and mobile phones.

  • Use flexible layouts.
  • Use relative units such as %, rem, and vw.
  • Optimize images.
  • Test on multiple screen sizes.
  • Avoid fixed-width layouts.

A responsive website provides a better experience for all users.

Lesson 20

Media Queries

Media queries apply different styles based on the screen size. 

@media (max-width:768px){

.container{

padding:15px;

}

.cards{

flex-direction:column;

}

}

Media queries are one of the most important tools for creating mobile-friendly websites.

Lesson 21

CSS Variables

CSS variables let you store reusable values that can be used throughout your stylesheet. 

:root{

--primary:#2563eb;

--secondary:#1d4ed8;

}

button{

background:var(--primary);

}

Variables make large projects easier to manage and keep colors and spacing consistent.

Lesson 22

CSS Best Practices

Writing clean CSS makes your website easier to maintain, faster to load, and simpler to update.

  • Use meaningful class names.
  • Keep your CSS organized.
  • Avoid repeating the same styles.
  • Use external stylesheets.
  • Group related properties together.
  • Use CSS variables for colors and spacing.
  • Write mobile-first responsive layouts.
  • Remove unused CSS regularly.
  • Comment complex sections of code.
  • Test your website on multiple browsers.

FAQ

Frequently Asked Questions

Can I learn CSS without JavaScript?

Yes. CSS is independent of JavaScript and should be learned after HTML.

Is CSS difficult?

CSS is beginner-friendly. It becomes easier with practice and by building real projects.

Should I learn Flexbox before Grid?

Yes. Flexbox is generally easier to understand and is commonly used for one-dimensional layouts. CSS Grid is ideal for more complex two-dimensional page layouts.

Lesson 23

CSS Pseudo-Classes

Pseudo-classes define the special state of an element, such as when a user hovers over it or clicks it. 

a:hover{

color:#2563eb;

}

button:active{

transform:scale(.95);

}

input:focus{

border:2px solid #2563eb;

outline:none;

}

Common Pseudo-Classes

  • :hover
  • :focus
  • :active
  • :visited
  • :first-child
  • :last-child
  • :nth-child()
Lesson 24

CSS Pseudo-Elements

Pseudo-elements style specific parts of an element. 

h1::after{

content:" 🚀";

}

p::first-letter{

font-size:32px;

font-weight:bold;

}

p::first-line{

color:#2563eb;

}

Pseudo-elements are useful for decorative effects without changing HTML.

Lesson 25

CSS Gradients

Gradients create smooth transitions between two or more colors. 

.hero{

background:linear-gradient(

135deg,

#2563eb,

#7c3aed

);

}

Gradient Types

  • linear-gradient()
  • radial-gradient()
  • conic-gradient()
Lesson 26

Box Shadow

The box-shadow property adds depth to cards, buttons, and images. 

.card{

box-shadow:

0 10px 30px

rgba(0,0,0,.15);

}

Shadows make interfaces appear more modern and visually appealing.

Lesson 27

Text Shadow

Text shadows add subtle depth to headings. 

h1{

text-shadow:

2px 2px 5px

rgba(0,0,0,.25);

}
Lesson 28

Overflow

Overflow controls what happens when content exceeds the size of its container. 

.box{

height:200px;

overflow:auto;

}

Overflow Values

  • visible
  • hidden
  • scroll
  • auto
Lesson 29

Z-Index

The z-index property controls which positioned elements appear in front of others. 

.modal{

position:fixed;

z-index:999;

}

Higher z-index values place an element above lower values, provided the elements are positioned.

Lesson 30

CSS Filters

CSS filters apply visual effects to images and other elements. 

img:hover{

filter:grayscale(100%);

}

.logo{

filter:drop-shadow(

0 5px 10px

rgba(0,0,0,.2)

);

}

Popular Filters

  • blur()
  • brightness()
  • contrast()
  • grayscale()
  • invert()
  • opacity()
  • saturate()
  • drop-shadow()
Lesson 31

CSS Performance Tips

Optimized CSS improves loading speed and creates a smoother user experience.

  • Keep stylesheets organized.
  • Remove unused CSS.
  • Minify CSS for production.
  • Avoid overly complex selectors.
  • Optimize web fonts.
  • Use modern layout techniques like Flexbox and Grid.
  • Compress images before using them.
  • Limit heavy animations on large pages.
